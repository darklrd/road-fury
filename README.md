# 🏎️ Road Fury

High-speed combat racing in your browser.

![Screenshot](screenshot.png)

## Play

```bash
pnpm install
pnpm dev
```

Open the URL shown in your terminal. Press any key to start, then:

| Key | Action |
|-----|--------|
| W / ↑ | Accelerate |
| S / ↓ | Brake |
| A / ← | Steer left |
| D / → | Steer right |

## Build

```bash
pnpm build
```

Output goes to `dist/`.

## Tech

- [Three.js](https://threejs.org/) for 3D rendering
- TypeScript (strict mode)
- Vite for dev/build
